FavoritesViewController

final class FavoritesViewController : UIViewController, ControllerConfiguration

Manages the view of Favorite items. Users can choose a Favorite by tapping it in order to apply the Favorite settings. The user may long-touch on a Favorite to bring up an editing panel.
