SoundFont

public final class SoundFont : NSObject, NSCoding

Representation of a sound font library. NOTE: all sound font files must have ‘sf2’ extension.

  • library

    Mapping of registered sound fonts. Add additional sound font entries here to make them available to the SoundFont code. NOTE: the value of this mapping is manipulated by the Python script catalog.py found in the Extras folder. In particular, it expects to find the -BEGIN- and -END- comments.
