Detailed Description

The source of an SF2 modulator.

There are two types:

  • general controller
  • MIDI continuous controller (CC)

The type is defined by the CC flag (bit 7). The general type points to various common MIDI values as a source of modulation, while the CC type allows for a broader collection of controllers.
