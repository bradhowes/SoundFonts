Detailed Description

Interpolatable index into a CanonicalBuffer.

Maintains two counters, an integral one (size_t) and a partial one (double) that indicates how close the index is to a sample index. These two values are then used by SampleBuffer routines to fetch the correct samples and interpolate over them.

Updates to the index honor loops in the sample stream if allowed. The index can also signal when it has reached the end of the sample stream via its finished method.
