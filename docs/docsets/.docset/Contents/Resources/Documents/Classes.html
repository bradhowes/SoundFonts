  • SoundFontsViewController

    View controller for the SoundFont / Patches UITableView combination. Much of the UITableView management is handled by specific *DataSource classes. This controller mainly serves to manage the active Patch state, plus the switching between normal Patch table view display and Patch search results display. Apart from the adopted protocols, there is no API for this class.

    Perhaps this should be split into two, one for a soundfont view, and one for the patches view.
