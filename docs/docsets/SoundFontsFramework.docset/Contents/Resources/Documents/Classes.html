    View controller for the SoundFont / Patches UITableView combination. Much of the UITableView management is handled by specific *DataSource classes.   • SettingsManager

    Manages access to user settings. Relies on UserDefaults for the actual storage. Originally, all settings were stored in the standard UserDefaults collection. However, with the introduction of the AUv3 extension, settings are now stored in a shared UserDefaults collection called “group.com.braysoftware.SoundFontsShare”. To work with older app installs, the manager will fall back to standard collection if a setting does not exist in the shared collection.
